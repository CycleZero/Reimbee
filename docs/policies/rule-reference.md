## 超标处理分级

| 级别 | 含义 | 处理方式 |
|:--:|------|---------|
| **pass** | 完全合规 | 正常提交，无需额外操作 |
| **warning** | 存在关注项，但不阻止提交 | 提示用户确认，审批人可见 warning 标记 |
| **error** | 严重违规 | 阻止提交，引导用户修正；超标部分不予报销 |

### warning 类规则
- 超标 <10% 且提交合理解释 → 审批人确认后可放行
- 无合理解释 → 升级为 error
- 示例：住宿 ¥370（超标 5.7%），注明"旺季酒店普涨" → warning

### error 类规则
- 票据过期（>90天）
- 人均消费超标 >10%
- 虚假/重复发票
- 单次金额超审批权限
